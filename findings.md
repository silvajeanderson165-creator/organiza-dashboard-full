# Pesquisas, Descobertas e Restrições

- Projeto: Task Manager MERN
- Base: `referencia.md`
- Data Schema de User e Task foram fixados em Constituição (`gemini.md`).

## Regras Fixadas na Descoberta:
- Sem Integrações externas de notificação (apenas Auth JWT standard).
- Utilizar MongoDB Local como base padrão de dados.
- O Frontend e Backend serão hospedados na mesma estrutura Vercel para deploy em produção.
- Visual Vibrante, sombras e animações requisitadas. Substituir o uso clássico de create-react-app (sugerido pela reference) para Vite focado em performance.
